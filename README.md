# UK Inflation VECM Analysis (2000–2025)
Econometric analysis of UK inflation dynamics (2000–2025) using VECM and time-series diagnostics in Stata.

This project analyzes the short-run and long-run determinants of UK inflation using a Vector Error Correction Model (VECM). The study uses monthly macroeconomic data from 2000 to 2025 and examines how monetary policy and energy prices influence inflation dynamics in the United Kingdom.

This work was completed as an academic project as part of the MSc Financial Management program at the University of Birmingham.

The analysis focuses on three key macroeconomic variables:
- Bank of England interest rates
- Unemployment rate
- Brent crude oil prices

The project applies time-series econometric techniques to identify long-run equilibrium relationships and short-run adjustments in inflation.

# Tools and Software

- Stata
- Microsoft Excel
- Time-Series Econometrics

# Methodology

The following econometric techniques were applied:

1. **Augmented Dickey-Fuller (ADF) Tests** 
   Used to test for stationarity in the time-series data.

2. **Johansen Cointegration Test**  
   Used to determine whether a long-run equilibrium relationship exists between inflation and the explanatory variables.

3. **Vector Error Correction Model (VECM)**  
   Used to estimate both long-run relationships and short-run dynamics among the variables.

4. **Lagrange Multiplier (LM) Test**  
   Used to check for serial correlation in the residuals and ensure model validity.

## Key Findings

- Brent crude oil prices have a significant long-run impact on UK inflation.
- Interest rates and unemployment were found to be statistically insignificant in the long-run equilibrium relationship.
- Inflation adjusts gradually toward equilibrium at approximately **4.88% per month**.
- Short-run dynamics show significant effects from lagged inflation, interest rates, unemployment, and oil price shocks.
